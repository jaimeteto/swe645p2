Course Overview

This course provides a detailed study of software components and component-based systems. Students will gain hands-on experience in building large-scale containerized web applications on AWS, with strong emphasis on DevOps engineering skills. Topics include Docker, Kubernetes, Jenkins, CI/CD pipelines, Helm, ArgoCD, Terraform, Ansible, React.js, Flask REST APIs, AWS Lambda, and more.
